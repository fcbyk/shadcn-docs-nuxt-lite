# shadcn-docs-nuxt-lite

A lite version of shadcn-docs-nuxt.

Some features and configurations have been removed to simplify daily usage and focus on writing Markdown docs.

Built on top of shadcn-docs-nuxt.

Thanks to the original project and contributors.
